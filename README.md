This is the code implementation for the paper titled: “Transformer-Based Local-Global Guidance for Image Captioning”
### Data Preparation
- The MSCOCO dataset is available at: https://cocodataset.org/#download
- The Flickr30k dataset is available at: https://www.kaggle.com/datasets/hsankesara/flickr-image-dataset
### Training & Evaluation
- To train/test the model, run the following command:
````
python train.py 
````
````
python evaluate.py 
````
